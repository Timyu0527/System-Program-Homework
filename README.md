# SIC/XE Assembler
## Environment
Platform: Ubuntu 22.04 LTS <br>
Language: C++17
## Usage
**Build** <br>
Build the assembler.
```
make
```
**Development** <br>
Build and run the assembler.
```
make dev
```
**Clean object file** <br>
Remove all object file and a.out.
```
make clean
```
**Run**
```
./a.out
<source file>
```
